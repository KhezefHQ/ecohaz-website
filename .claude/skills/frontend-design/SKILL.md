---
name: frontend-design
description: Create distinctive, production-grade frontend interfaces with high design quality. Use when building web components, pages, artifacts, or applications. Avoids generic AI aesthetics.
---

When building any frontend, choose a BOLD aesthetic direction. Avoid generic AI aesthetics like Inter/Roboto fonts, purple gradients, and predictable layouts.

Use distinctive typography, strong color palettes, meaningful animations, and intentional layouts. Every design should be memorable and specific to its context.
